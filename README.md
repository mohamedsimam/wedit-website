# Wedit Agency Website

**Wedit** is a premium, high-performance web development agency website designed for students, freelancers, and local business owners. Built with a modern tech-theme, it features advanced animations, glassmorphism, and a robust architecture.

## 🔥 Key Features
- **Premium Tech Aesthetic**: Dark theme with deep indigo glows and futuristic glassmorphic UI.
- **4 Custom Solutions**: Specialized paths for Portfolios, WhatsApp Stores, E-commerce, and Web+App solutions.
- **Innovative Process**: A clear 5-stage roadmap from Discovery to Scaling.
- **Advanced Animations**: Powered by `Framer Motion` for a silky-smooth user experience.
- **Lead Management**: Pre-configured Supabase client for seamless client intake.
- **Highly Responsive**: Optimized for all devices and screen sizes.

## 🛠️ Tech Stack
- **Framework**: [React](https://react.dev/) + [Vite](https://vite.dev/)
- **Animations**: [Framer Motion](https://www.framer.com/motion/)
- **Icons**: [Lucide React](https://lucide.dev/)
- **Database**: [Supabase](https://supabase.com/)
- **Styling**: Vanilla CSS with custom design tokens.

## 🚀 Getting Started
1. **Clone the Repo**:
   ```bash
   git clone https://github.com/mohamedsimam/wedit-website.git
   ```
2. **Install Dependencies**:
   ```bash
   npm install --legacy-peer-deps
   ```
3. **Run Locally**:
   ```bash
   npm run dev
   ```
